# Fortune-Teller
Fortune Teller is a C++ console application that generates random fortunes across Love, Career, Daily Life, and Health. Built as a beginner project, it demonstrates core C++ concepts including arrays, loops, switch statements, user input/output, and random number generation through an interactive command-line experience.

---

## 📖 Project Overview

The Fortune Teller allows users to choose from four different fortune categories:
* ❤️ Love
* 💼 Career
* 🌞 Daily Life
* 🌿 Health

Based on the selected category, the program randomly displays one fortune from a predefined set of messages. Users can continue receiving fortunes until they choose to exit the application.

## ⚙️ How It Works

1. The program displays a welcome message.
2. The user selects a fortune category.
3. A random fortune is chosen from the selected category.
4. The fortune is displayed on the screen.
5. The user can choose to receive another fortune or exit the program.
